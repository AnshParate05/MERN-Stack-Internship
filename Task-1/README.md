# Task 1 - MERN Stack Project

This is a full-stack MERN (MongoDB, Express, React, Node.js) application.

## Project Structure

```
Task-1/
├── backend/          # Node.js + Express server
├── frontend/         # React application
└── README.md
```

## Setup Instructions

### Backend Setup
1. Navigate to backend folder: `cd Task-1/backend`
2. Install dependencies: `npm install`
3. Create `.env` file with your environment variables
4. Start server: `npm start`

### Frontend Setup
1. Navigate to frontend folder: `cd Task-1/frontend`
2. Install dependencies: `npm install`
3. Start development server: `npm start`

## Features
- RESTful API with Express
- MongoDB database integration
- React frontend with modern hooks
- Environment variable configuration
- CORS enabled for frontend-backend communication
